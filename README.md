Basic Electron Database Boilerplate
-----------------------------------
v1.0.0

---------------------
Included
-----------------

 - electron: ^1.8.2
 - electron-packager: ^11.0.1
 - electron-reload: ^1.2.2
 - knex: ^0.14.4
 - mysql2: ^1.5.2
 - yaml: ^0.3.0

Installation
-------------
```
git clone https://github.com/myazarc/electron-db-boilerplate.git your-working-dir
cd your-working-dir
npm install
npm run start
```
